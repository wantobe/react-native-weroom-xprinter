# react-native-weroom-xprinter

芯烨热敏打印机SDK封装，只有实现了安卓平台打印功能，其他功能请自行实现

## Installation

```sh
npm install react-native-weroom-xprinter
```

## Usage

```js
import { multiply } from 'react-native-weroom-xprinter';

// ...

const result = multiply(3, 7);
```
